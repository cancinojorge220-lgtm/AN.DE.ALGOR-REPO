# a) Determine una ruta entre dos ciudades seleccionadas.

    Se seleccionaron las ciudades Lima y Piura. Para encontrar una ruta se utilizó el algoritmo BFS (Búsqueda en Anchura), el cual explora primero los nodos más cercanos antes de continuar con los demás. La ruta encontrada fue:

    Lima → Trujillo → Chiclayo → Piura

# b) Implemente un algoritmo que permita obtener un Árbol de Recubrimiento Mínimo.

    Se implementó el algoritmo de Prim. Este algoritmo inicia desde una ciudad y va agregando las conexiones de menor costo hasta conectar todas las ciudades sin formar ciclos. De esta manera se obtiene una red de distribución con el menor costo posible.

# c) Explique cómo esta solución ayudaría a reducir costos logísticos.

    El Árbol de Recubrimiento Mínimo permite conectar todas las ciudades utilizando únicamente las rutas necesarias y de menor costo. Esto ayuda a la empresa a reducir gastos de transporte, combustible, mantenimiento de vehículos y tiempo de operación. Además, evita conexiones redundantes entre ciudades, logrando una red logística más eficiente y económica.
